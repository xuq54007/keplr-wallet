# Layout

Layout을 위한 컴포넌트들이 있다.<br />
Component를 통해서 Page를 구축할 때 Styling을 직접 보지 않고도<br />
Layout Component들의 구성을 보고 대략적인 Page의 모습을 알 수 있게 하기 위해서 사용된다<br />

# 기본 구성

화면은 기본적으로 vertical base로 위에서 아래로 이루어지는 구성이라고 가정한다<br />
css에서 width, height를 설정하지 않을때 default 값은 auto가 되는데<br />
auto일때 width는 parent의 남은 공간을 모두 차지하고 height는 자식의 height로 설정된다<br />
이런 방식과 layout 구성을 이용해서 component에서는 width 자체를 직접 설정하는 방식을 될 수 있는한 피하고 layout에 의해서 width가 결정되도록 한다<br />
